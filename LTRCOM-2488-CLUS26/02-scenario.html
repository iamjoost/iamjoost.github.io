Scenario

In this lab activity, you will learn how to configure UCS X-Series via Intersight. This demonstration provides you access to a dedicated physical Cisco UCS X-Series system. You access the UCS X-Series system using a SSO Cisco Intersight account that connects to the Cisco UCS X-Series system as a target. The lab takes you through configuring the UCS X-Series system manually, using the Cisco Intersight GUI.
