Learning Objectives

Upon completion of this lab, you will be able to:

  • Understand UCS Domain-, Chassis and Server profiles for UCS X-Series
  • Look at the right places for information
  • Create Templates, Profiles and Policies
  • Know best practices for UCS X-Series policies.
  • Install OS on a server.
  • Create your Metrics and put it on a Dashboard
  • Work with the Intersight API Client to get information.
