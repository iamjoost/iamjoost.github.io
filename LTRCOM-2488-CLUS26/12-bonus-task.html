Bonus Task

This will be a task without any screenshots, just to prove how easy it is to upgrade server firmware.

DO NOT UPGRADE THE FABRIC INTERCONNECT FIRMWARE.

Upgrade server firmware

At this moment the server firmware is 5.2(0.230092)

On the right of your server, click on the three dots and select Upgrade Firmware.

Follow the steps and select a firmware that is preferable higher.

Firmware versions can be set in a policy.
